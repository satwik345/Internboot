## Sales Analysis and Forecasting

###  Project Overview
The **Sales Analysis and Forecasting Project** focuses on **Exploratory Data Analysis (EDA)** and **basic time-series forecasting** using real-world retail sales data.  
The dataset is sourced from the Kaggle competition **‚ÄúStore Sales ‚Äì Time Series Forecasting‚Äù**, which contains historical sales data across multiple stores and product families.

---

##  Project Objectives

### **Task 1: Exploratory Sales Analysis**
**Description:**  
Perform exploratory data analysis (EDA) on sales data to understand:
- Sales trends
- Seasonality patterns
- Missing values and data quality issues  

---

### **Task 2: Simple Sales Forecast with Moving Average**
**Description:**  
Implement a basic forecasting approach using moving averages to:
- Smooth sales trends
- Compare actual sales with forecasted values
- Analyze short-term and long-term patterns  

---

### **Task 3: Linear Regression for Sales Prediction**
**Description:**  
Build a simple machine learning model to predict future sales using:
- Time-based features
- Promotions and holidays  
- Train-test data split and evaluation
